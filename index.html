Footer

OODLIB Footer directive demo

You can simply edit footer data in CoffeeScript.

TAG
<cfooter>
ATTRIBUTE:
info: It is used to connect your data. You can use three keys in data:
  • copyright [String]
  • version [String]
  • websites [Array] link name -> key, link -> value
